About this Course

Web Design is an introductory one semester course that surveys the concepts and skills necessary to develop a web site for a small business or organization. Subjects covered:

  • Designing and Planning Web Pages
  • Creating Web Pages with HTML
  • Formatting with Style Sheets (CSS)
  • Graphics
  • Scripting
  • Client Website Project

Unit 2

In Unit 2, we learned the basics of implementing HTML. We learned how to add various types of text and content on our website. The first thing that we needed to add to our website were the common and essential tags to be able to add content that we wanted. After setting up the tags that we needed to have correct formatting. We implemented the content that we wanted to display through text at first. We moved on to integrating videos into our website for our video page. We also have links to external and internal sites that can be accessed throughout the pages.

Unit 4

In Unit 4, we began to learn about adding graphics to our websites. First, we learned about how to use Paint.NET and other image processers to create images. Then we learned how to implement these images into our websites, and use them as links to other sections of the website using the 'a' tag. Finally, we created a banner for our websites by learning how to select portions of images and move them around to our pleasing. We added that image to the top of our website (mine is commented out) as a banner.

Unit 5

In unit 5 we learned about the basics of using Javascript. We learned about the differences between server side scripting (PHP, Python) and client side scripting (Javascript). Within Javascript, we learned how to create functions and variables and use the DOM (Document Object Model). We used this knowledge to create clocks that read out the time. We then, finally, learned how to add this script to our program and apply it to the DOM to create a clock and 2 buttons (one for the alert and one to show/hide the clock).
